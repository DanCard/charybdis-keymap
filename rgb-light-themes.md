# RGB Light Themes

The keyboard cycles through RGB modes every 30 seconds when `KC_RGB_AUTO` is enabled. When the mode changes, the mode number is displayed by flashing the corresponding number keys white.

| 9 | COLORBAND_PINWHEEL_VAL | Pinwheel value |
| 10 | COLORBAND_SPIRAL_SAT | Spiral saturation |
| 11 | COLORBAND_SPIRAL_VAL | Spiral value |
| 12 | CYCLE_ALL | All keys cycle together |
| 13 | CYCLE_LEFT_RIGHT | Rainbow sweep left/right |
| 14 | CYCLE_UP_DOWN | Rainbow sweep up/down |
| 15 | CYCLE_OUT_IN | Rainbow from edges to center |
| 16 | CYCLE_OUT_IN_DUAL | Dual rainbow waves |

**Note:** Actual mode numbers depend on which effects are enabled in your firmware. Not all modes may be available.

## Quick Access Keys (Layer 2)

| Key | Mode |
|-----|------|
| Q | CYCLE_LEFT_RIGHT (Rainbow) |
| W | SPLASH (Reactive) |
| E | JELLYBEAN_RAINDROPS |
| R | CYCLE_SPIRAL |
| T | RAINBOW_MOVING_CHEVRON |

## Controls

- **RM_NEXT**: Cycle to next mode
- **KC_RGB_AUTO**: Toggle auto-cycle (every 30 seconds)
